Demo breve — TrackerBot Gaming

Hola, soy Luis. Hoy les muestro TrackerBot Gaming en 60 segundos.

1) Objetivo (5s)
- Monitorizar precios de juegos en Instant Gaming y enviar alertas por Telegram.

2) UI (10s)
- Panel principal con tarjetas para FIFA 26, GTA V y Call of Duty: Black Ops.
- Controles: Iniciar, Detener, Comprobar ahora, Descargar instantánea.

3) Simulación (20s)
- Pulso `Simular alerta` para demostrar una alerta sin depender del scraping.
- Se mostrará la notificación en pantalla y se registrará en el historial.

4) Telegram (15s)
- Rellenas Bot Token y Chat ID → `Probar Telegram` → mensaje confirmado en tu chat.

5) Cierre (10s)
- `Descargar instantánea` para guardar el estado actual.
- Limitación: scraping en cliente usa proxies públicos; para producción usar backend.
